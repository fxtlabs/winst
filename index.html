API Overview - Winst ()


Track your trading activities, holdings, and realized gains in multiple currencies.


winst.core

by Filippo Tampieri <fxt@fxtlabs.com>
 Detailed API documentation
The engine that processes trading activities to compute resulting
holdings and realized gains/losses.
Public variables and functions: apply-activity compute-holdings compute-holdings-seq compute-realized-gains convert-activities convert-activity dated-before? dated-within? merge-by-date sort-by-date

winst.currency

by Filippo Tampieri <fxt@fxtlabs.com>
 Detailed API documentation
Services for working with different currencies, including parsing
exchange rate CSV files and finding out the exchange rate for a
given day.
Public variables and functions: currency-keyword currency-name get-exchange-rate-lookup

winst.dsl

by Filippo Tampieri <fxt@fxtlabs.com>
 Detailed API documentation
An embedded domain specific language to easily describe trading activities.
Public variables and functions: activities activity get-security-uid

winst.handlers

by Filippo Tampieri <fxt@fxtlabs.com>
 Detailed API documentation
Request handlers for the trading activity, holdings, and realized
gains/losses of an account over a specified period of time.
Public variables and functions: activities-handler gains-handler holdings-handler

winst.presentation

by Filippo Tampieri <fxt@fxtlabs.com>
 Detailed API documentation
The presentation layer of the winst server. It renders HTML reports
for the trading activity, holdings, and realized gains/losses of an
account over a specified period of time.
Public variables and functions: render-activities render-error render-gains render-holdings render-not-found

winst.routes

by Filippo Tampieri <fxt@fxtlabs.com>
 Detailed API documentation
Maps URLs to request handlers to report trading activity, holdings,
and realized gains/losses for an account over a specified period of time.
Public variables and functions: main-routes

winst.securities

by Filippo Tampieri <fxt@fxtlabs.com>
 Detailed API documentation
Services for working with securities. It manages a list of all the
companies listed on the NASDAQ, NYSE, and AMEX exchanges. The official
lists are available for download as CSV files from the NASDAQ web site
at http://www.nasdaq.com/screening/company-list.aspx
Public variables and functions: lookup-security lookup-security-symbol securities-by-uid security-qualified-symbol security-uid
